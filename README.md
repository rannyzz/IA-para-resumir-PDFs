# PDF Mentor

Transforme PDFs em resumos inteligentes usando Inteligência Artificial local.

## 🚀 Sobre o projeto

O StudyMind AI é uma aplicação web que permite enviar arquivos PDF e gerar resumos automáticos utilizando IA.

O projeto foi desenvolvido para auxiliar estudantes a revisarem conteúdos de forma rápida e eficiente, transformando documentos extensos em informações organizadas e fáceis de estudar.

## ✨ Funcionalidades

- 📄 Upload de arquivos PDF
- 🤖 Geração automática de resumos com IA
- 📝 Extração de texto dos documentos
- 🎨 Interface simples e intuitiva
- 🔒 Processamento local utilizando Ollama e Llama 3.2

## 🛠️ Tecnologias Utilizadas

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express
- Multer
- pdf-parse
- Ollama

### IA
- Llama 3.2

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/rannyzz/IA-para-resumir-PDFs.git
```

Entre na pasta:

```bash
cd IA-para-resumir-PDFs
```

Instale as dependências:

```bash
npm install
```

Instale e execute o Ollama:

```bash
ollama pull llama3.2
```

Inicie o servidor:

```bash
node backend/server.js
```
